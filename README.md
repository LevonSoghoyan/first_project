#First Project 
Task Pilot is a Command Line Interface.
It allows users to manage  tasks directly 
from the terminal using simple commands like 
addig,listing,completing, and cleaning tasks.
##Features
- ADD tasks
- View complete tasks
- View all tasks
- Change task status
- Clear task list
